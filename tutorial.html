2Create MCP Server

‚è±Ô∏è Time Required: 15 minutes
üí° What You'll Learn: Building a basic MCP server

Create Server File

Create an index.js file in your mcp-server directory with the following structure:

mcp-server/index.js 
import { Server } from '@modelcontextprotocol/sdk/server/index.js';
import { StdioServerTransport } from '@modelcontextprotocol/sdk/server/stdio.js';

// Initialize the MCP server
const server = new Server(
  {
    name: 'task-manager-server',
    version: '1.0.0',
  },
  {
    capabilities: {
      tools: {},
    },
  }
);

// Start the server
const transport = new StdioServerTransport();
await server.connect(transport);

console.log('Task Manager MCP Server running');

5Connect Frontend to MCP

‚è±Ô∏è Time Required: 10 minutes
üí° What You'll Learn: Connecting React frontend to MCP server

Create API Client

Create a service to communicate with your MCP server:

react-client/src/services/mcpClient.js 
class MCPClient {
  constructor() {
    this.baseUrl = 'http://localhost:3001';
  }

  async getTasks() {
    const response = await fetch(`${this.baseUrl}/api/tasks`);
    return await response.json();
  }

  async addTask(task) {
    const response = await fetch(`${this.baseUrl}/api/tasks`, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify(task),
    });
    return await response.json();
  }
}

export default new MCPClient();
